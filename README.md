# SmartConf Claw

Desktop research copilot built with React, TypeScript, TailwindCSS, and Tauri v2.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:1420](http://localhost:1420) for web UI testing.

## Tauri Dev

```bash
npm run tauri:dev
```

## Build

```bash
npm run build
```
